# ストップウォッチアプリ 使い方ガイド

マジシャン用ストップウォッチアプリの操作マニュアルです。お客様の前で自然に使えるよう、基本操作から仕掛けの設定まで説明します。

---

## 1. ホーム画面への追加（インストール）

1. iPhoneのSafariでアプリのURLを開く
2. 共有ボタン（四角に上矢印）をタップ
3. 「ホーム画面に追加」を選択
4. 名前は「ストップウォッチ」のまま「追加」をタップ

ホーム画面に黒地にオレンジのストップウォッチアイコンが追加されます。タップするとアドレスバーなしの全画面表示で起動します。純正の時計アプリと並べて置いても違和感がありません。

---

## 2. 基本操作

### 画面の見た目

iPhone純正の「時計」アプリのストップウォッチ画面を再現しています。

- 上部: 経過時間（分:秒.コンマ以下2桁）
- 中央下: ラップ／リセットボタン（左）、開始／停止ボタン（右）
- 下部: タブバー（世界時計・アラーム・ストップウォッチ・タイマー）

### 操作の流れ

| ボタン | 状態 | 動作 |
|--------|------|------|
| 開始（緑） | 初期・停止中 | タップで計測開始 |
| 停止（赤） | 計測中 | タップで計測停止 |
| ラップ（グレー） | 計測中 | タップでラップを記録 |
| リセット | 停止中 | タップで00:00.00に戻す |

計測中に停止→開始すると、停止していた時刻から続けて計測されます（0には戻りません）。

### ラップ機能

- 計測中に左ボタン（ラップ）を押すと区間タイムが記録されます
- 新しいラップが一覧の一番上に追加されます
- ラップが2つ以上記録されると、最も速い区間が緑、最も遅い区間が赤で表示されます
- リセットするとラップ一覧もクリアされます

## 3. 仕掛けの設定（隠し設定画面）

### 開き方

下部タブバーの**「世界時計」アイコンを素早く3回タップ**します。1.5秒以内に3回タップする必要があります。

「表示設定」という画面が開きます。お客様に見られても「時計の細かい設定画面」にしか見えないデザインにしてあります。

### 数値補正（仕掛けの本体）

ここに数字を入力すると、その数字が「種」になります。

**仕組み**

ストップウォッチを止めた瞬間、表示される「秒」と「コンマ以下」を足すと、必ずここで設定した数字になります。

例: 「105」と設定した場合
- 17.46で止まったように見えても、実際は「46 + 59 = 105」になるよう、コンマ以下が自動調整されて「17:46.59」と表示されます

**設定方法**

1. 隠し設定画面を開く
2. 「数値補正」欄に数字を入力（0〜158の範囲）
3. 「保存」をタップ

**注意点**

- 入力した数字が大きすぎる・小さすぎる場合、調整が効かず実際の時間がそのまま表示されることがあります（秒の値によっては調整しきれない組み合わせがあるため）。事前に何度かテストして安定する数値を確認してください
- 欄を空欄にする、または「0」にすると仕掛けがオフになり、普通のストップウォッチとして動作します

### 開始条件（仕掛けが発動するタイミング）

最初から仕掛けが効いていると不自然な場合に使う設定です。「いつから仕掛けを効かせるか」を選べます。

| モード | 説明 |
|--------|------|
| なし | 数値補正を設定したら毎回仕掛けが効く（デフォルト） |
| 回数 | 停止ボタンをN回押した後から仕掛けが効き始める |
| 時間 | 最初に計測を開始してからN秒経過後に仕掛けが効き始める |

**回数モードの例**

「3」に設定した場合、1回目・2回目の停止は普通の時間が表示され、3回目以降の停止から仕掛けが発動します。「何度か普通に止めて見せてから、最後にピタリと合わせる」という演出に使えます。

**時間モードの例**

「60」に設定した場合、最初に「開始」を押してから60秒間は普通の時間が表示され、60秒を過ぎてからの停止で仕掛けが発動します。

**設定方法**

1. 隠し設定画面を開く
2. 「開始条件」で「なし」「回数」「時間」のいずれかを選ぶ
3. 「回数」または「時間」を選んだ場合、下に出てくる欄に数値を入力
4. 「保存」をタップ

**注意点**

- リセットボタンを押すと、停止回数・経過時間のカウントは0に戻ります（仕掛けの設定自体は消えません）
- 「回数」「時間」どちらか一方しか選べません

### キャンセル

設定を変更したくない場合は「キャンセル」をタップするか、画面の暗い部分をタップすると、変更を保存せずに閉じられます。

## 4. 演技前の準備チェックリスト

1. ホーム画面のアイコンからアプリを起動する
2. 世界時計を3回タップして設定画面を開く
3. 今回の演技で見せたい数字を「数値補正」に入力する
4. 必要なら「開始条件」を設定する（最初から決めておきたいなら「なし」のままでOK）
5. 保存して、リセットボタンで00:00.00に戻っているか確認する
6. 一度自分で開始→停止のテストをして、狙った数字になるか確認する

## 5. よくある質問

**Q. 仕掛けが効かない**
A. 数値補正の値と実際の秒数の組み合わせによっては、コンマ以下が0〜99の範囲に収まらず仕掛けが効かないことがあります。設定値を見直すか、数値補正を変更してください。

**Q. お客様に設定画面を見られそう**
A. 「世界時計」タブを3回連続タップしないと開かないため、通常の操作で誤って開くことはほぼありません。万が一見られても、ラベルは「表示設定」「数値補正」など無害な表記にしてあります。

**Q. 計測中に画面を切り替えたら止まりますか**
A. ブラウザ／ホーム画面アプリを閉じたりバックグラウンドに回すと、再度開いたときに計測が止まっている場合があります。演技中は他のアプリに切り替えないようにしてください。
