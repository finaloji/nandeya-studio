# magcharge（鼻で充電・MagSafeフォース） 仕様書

**ステータス：構想・壁打ち段階（ステップ0〜7完了 → ドラフト作成済み。実装は /sprint で進める）**

コードネーム: `magcharge`（表向きは名前が出ないため、正式名は後で変更可・※要相談）

---

## マジックの全体像（確定）

iPhoneの充電ケーブルの先を自分の鼻の穴に挿すと、「プンっ」という音とともに
画面に **MagSafe充電開始そっくりの全画面アニメ**（中央の円形バッテリー％＋緑の波紋）が
ブワッと現れ、「鼻から充電された!?」と観客を笑わせる純粋ギャグ一発芸。

- 表向きは **ただのロック画面**（ユーザー自身のロック画面スクショを全画面表示）。
- 発動は **時間（タイマー）自動**。マジシャンがこっそりアームし、秒数を体で合わせて
  ケーブルを鼻に挿した瞬間にアニメが出るよう仕込む。
- 掴み〜本番は一体（別フォースには繋げない。純粋ギャグ一本）。

## 演出の流れ（確定）

1. **前フリ**：「最近スマホ、どこでも充電できるの知ってます？」等。画面は寝てるロック画面。
2. **アーム**：演技の流れの中で、画面の隠しエリアをこっそり操作してカウント開始（無反応）。
3. **仕込み**：充電ケーブルを取り出し、コネクタ側を自分の鼻の穴へ近づける。
4. **発動**：頭でカウントした秒数どおり、挿した瞬間にMagSafeアニメ＋「プンっ」音＋擬似輝度フラッシュ。
5. **クライマックス**：中央のバッテリー％が**急速でグングン上昇**。観客に見せる。
6. **オチ**：数秒で**一気に100%まで到達**してドヤる（「鼻で急速充電完了」）。抜くとロック画面に戻る。

## フォースの仕組み（確定）

- **タイマー自動発動（隠しアーム式）**
  - アプリを開いただけでは何も起きない（開いた瞬間の自動カウントは誤爆リスクで却下）。
  - マジシャンが**隠しエリアを長押し（0.5〜1秒）でアーム** → カウント開始。
  - 設定秒数（隠し設定で可変）経過後、自動でMagSafe演出をON。
  - マジシャンは秒数を体で覚え、ケーブルを鼻に挿すタイミングを合わせる（発動直前サインは無し・確定）。

## UI・通常動作（確定）

- 全画面に**ユーザー登録の壁紙画像**を表示し、その上に**ライブの日付・時刻**を本物のロック画面そっくりに描画する。一見ただのロック中iPhone。
  - 登録画像は**時計が写っていない壁紙**を前提とする（時計の二重表示を防ぐため。Re:Time と同方式）。
  - 日付・時刻は実時刻で自動更新（Re:Time の `.date`/`.clock` 表示・`_tick`/250ms更新を流用）。
- ネット非依存で演出が完結（後述）。
- 通常状態では隠しエリアを触っても**完全無反応**（色変化・ハイライト等を一切出さない）。

## 隠し操作・隠し設定（確定）

- **アーム**：画面の隠しエリアを長押し（誤爆しにくい操作）→ 無音・無反応でカウント開始。
- **隠し設定画面**（アクセス方法は実装時に確定・※要相談）で以下を調整：
  - 発動までの秒数（例 3〜10秒、可変）
  - ロック画面スクショの登録／差し替え
  - 音のON/OFF・音量など**細かい音設定**（確定）
  - 擬似輝度フラッシュのON/OFF（任意）

## ONサイン（確定）

- **発動直前サインは無し**。マジシャンは頭のカウントのみでタイミングを合わせる。
  （画面の光サインは覗かれ危険で却下、バイブサインも今回は不採用）

## クライマックス演出（確定・実機動画のフレーム分析準拠）

発動から約10.5秒で自動完結する6フェーズ構成：

1. **すりガラス化**（0〜0.6秒）：壁紙がボケて（backdrop-filter）時計が消える
2. **リング出現**（0.6〜1.4秒）：ぼんやりした円→輪郭が絞まって半透明白のリングに。緑アークは12時起点・開始%ぶん
3. **稲妻＋テキスト**（1.4〜1.9秒）：リング中央に緑の稲妻、下に「NN% 充電済み」がフェードイン
4. **じわじわ充電**（1.9〜9.2秒）：開始%（25〜45ランダム）→100%まで単調増加
5. **時計回りワイプ**（9.2〜9.9秒）：100%で0.2秒静止→リングが12時から時計回りにシュッと食われて消え、稲妻・文字フェードアウト
6. **フェード復帰**（9.9〜10.5秒）：すりガラスが解けて元のロック画面へ自動復帰・アーム解除

- タップでの緊急脱出は全フェーズで有効（即復帰・アーム解除）。
- 「プンっ」＝MagSafe吸着音風の効果音は 4-1 で追加予定。
- ※本物のiOSステータスバー・実機輝度はWebから操作不可のため全画面自作演出で再現（確定した技術制約）。

### 却下・変更履歴
- 初版（黒背景＋中央大%数字＋3.5秒急上昇）は実機動画と乖離があり 3-2 でリメイク済み。
- ※本物のiOSステータスバー右上の電池アイコン／稲妻マーク／実機の画面輝度は
  **Webアプリから操作不可**（確定した技術制約）。よって全画面の自作演出で丸ごと再現する方針。

## データの保存（確定）

- すべてローカル・**難読化キー**で保存（ネット非依存）。
  - 発動秒数（例 `_dcc`）
  - 音設定・輝度フラッシュ設定（例 `_tcm` 等）
- **ロック画面スクショ画像**はサイズが大きいため **IndexedDB** に保存（キーも難読化）。

## ライセンス管理（確定）

- 他マジックアプリと共通：Firebase Firestore ＋独自ID/パスコード方式、2台までのLRU端末ロック。
- 管理画面は `apps/stopwatch/admin.html` に集約（新規作成しない）。
- **一度認証したらオフラインでも発動可**（認証結果をキャッシュ）。演出自体はネット非依存。

## 難読化方針（共通ルール準拠）

- localStorageキー・IndexedDBキー・関数名は意味のない名前（`_dcc`, `_tcm`, `dcc()` 等）。
- UIラベルは「表示設定」「数値補正」などネタが割れない言葉。
- コードを読んでも「鼻で充電するギャグ」機能の存在が分からないようにする。

## スマホ最適化（共通ルール準拠）

- モバイルファースト／PWA対応（manifest.json、apple-touch-icon）。
- iOSのnumeric input対策として秒数設定等は決定ボタンを明示設置。
- 全画面表示・セーフエリア考慮（ノッチ/ホームインジケータ）。

## リスク検証結果（ステップ6・確定）

- **画面を覗かれたら**：常時ロック画面スクショなので無害。アームは完全無反応にする（対策済み）。
- **観客に触られたら・連打されたら**：
  - アームは長押しにして誤爆防止。
  - **発動後（アニメ再生中）は全タップ無効**（渡しても崩れない）。
- **ネットが切れたら**：演出・音・画像は全ローカル完結。認証はキャッシュしオフライン動作可。

### 検討して却下した案（再議論防止）

- **オチをBの掴み＋別フォース**：ネタが強い一発芸なので数字当て等を乗せると散らかる → 却下。
- **本物ステータスバー右上の電池/稲妻を光らせる**：iOSシステムUIでWebから操作不可 → 全画面自作演出に変更。
- **画面輝度を実際に上げる**：Safari/PWAに輝度APIなし → 擬似輝度フラッシュで代替。
- **タップでその場発動**：時間発動の方が演技中に画面を触らず自然 → 時間発動採用。
- **アプリを開いた瞬間から自動カウント**：前フリでモタつくと勝手に発動 → 隠しアーム式に。
- **発動直前の光サイン**：覗かれ危険 → サインなし（秒数カウント）に。
- **バッテリー診断アプリ風カバー**：MagSafeはロック画面に出るものなので、ロック画面風の方が自然 → 却下。

## スプリント一覧（案）

1. ロック画面スクショ全画面表示（IndexedDB登録・差し替え）＋PWA土台
2. 隠しアーム（長押し・無反応）＋可変タイマー→自動発動の骨組み
3. MagSafe風全画面アニメ（円形バッテリー％＋波紋）＋擬似輝度フラッシュ
4. 「プンっ」効果音＋音の細かい設定（ON/OFF・音量）
5. アニメ中の全タップ無効・抜き操作で停止などの誤爆対策
6. ライセンス認証組み込み（Firestore・2台LRU）＋オフラインキャッシュ、admin.html連携
7. 難読化仕上げ・実機調整（セーフエリア/タイミング）
