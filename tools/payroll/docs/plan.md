## 第1弾のスコープ

**打刻・勤怠一覧・修正承認まで。** 給与計算・コミッション・明細PDF・月次確定は第2弾以降。

## ファイル構成（予定）

| ファイル | 内容 |
|---|---|
| `index.html` | 管理者・社員向け（マスタ管理・勤怠一覧・修正申請・操作履歴） |
| `punch.html` | スタッフ向け打刻画面（トークン付きURLで開く） |
| `manifest.json` / `sw.js` | PWA対応 |
| `firestore.rules` | ロール別の権限（手動デプロイ） |

管理画面と打刻画面は利用者も使う頻度も違うため、HTMLを分ける。
スタッフがPWAとしてホーム画面に置くのは `punch.html`。
