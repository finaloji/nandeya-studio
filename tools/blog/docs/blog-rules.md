# 株式会社なんで屋 出張レポートブログ執筆ルール

## ブログの目的

- マジシャン派遣の実績を伝える
- 初めてマジシャンを呼ぶ担当者の不安を減らす
- どのようなイベントで依頼できるかを伝える
- 神戸・大阪・関西を中心とした検索流入につなげる
- 最終的に問い合わせにつなげる

## タイトル

以下の形式を基本とする。

【地域】会場名でイベント名のマジックショーを行いました～担当者名～

例：

【大阪市】KKRホテル大阪でマンション居住者懇親会のマジックショーを行いました～まさゆき～

タイトルには、可能な範囲で以下を含める。

- 地域
- 会場名
- イベント種別
- マジックショー
- 担当者名

## 冒頭

必ず以下の形式で始める。

こんにちは。株式会社なんで屋の〇〇です！

〇〇には、記事の担当マジシャン名を入れる。

その後、今回どこで、どのようなイベントに出演したのかを
2～3文で簡潔に説明する。

## 見出し

見出しは原則として以下の順番で固定する。

1. イベント概要
2. タイムスケジュール
3. 担当マジシャン
4. マジックショーの内容
5. 演出のポイント
6. 会場・お客様の反応
7. 同じようなイベントをご検討中の方へ

## イベント概要

以下を文章または見やすい形式で記載する。

- 開催日
- 地域
- 会場名
- イベント種別
- 来場者数
- 年齢層
- ショー形式
- 公演時間

入力されていない項目は推測しない。

## タイムスケジュール

入り時間、準備、本番、撤収など、
入力された内容を時系列で分かりやすく記載する。

すべての時刻が入力されていない場合は、
分かっている範囲だけで自然な文章にする。

## 担当マジシャン

出演したマジシャンやアシスタントを紹介する。

プロフィール情報が登録されている場合のみ使用する。
未登録の経歴や実績を作らない。

## マジックショーの内容

演目を単純に箇条書きするだけでなく、
ショー全体の流れが伝わるように自然な文章でつなげる。

例：

オープニングでは〇〇を行い、会場の皆様に注目していただきました。
続いて、お客様にもご参加いただく〇〇を披露しました。

演目の仕掛けや秘密は書かない。

## 演出のポイント

以下のような、今回の現場ならではの工夫を書く。

- 会場の広さへの対応
- お子様や高齢者への配慮
- ファミリー向けの進行
- 企業イベント向けのテンポ
- 観客参加型の演出
- 音響、照明、登場方法
- 会場レイアウトへの対応

入力されていない工夫を勝手に作らない。

## 会場・お客様の反応

実際に入力された反応をもとに書く。

「大盛り上がりでした」だけで終わらせず、
どの場面で、どのような反応があったかを具体的にする。

ただし、実際に確認できていない感想や発言は作らない。

## 同じようなイベントをご検討中の方へ

今回の事例と似たイベントを検討している方に向けて、
マジックショーを活用できることを案内する。

問い合わせへの誘導は、押し売り感を出さず、
相談しやすい文章にする。

例：

株式会社なんで屋では、会場の広さや参加人数、
年齢層に合わせたマジックショーをご提案しています。

初めてマジシャンを呼ばれる場合も、
演出内容や当日の流れからご相談いただけます。
お気軽にお問い合わせください。

## 文体

- 丁寧だが、堅すぎない
- 親しみやすく読みやすい
- 文章は長くなりすぎない
- 同じ語尾を3回以上続けない
- 「させていただきました」を多用しない
- 大げさすぎる表現を避ける
- 絵文字は原則使用しない
- 感嘆符は必要最小限にする

## URLスラッグ

短い英語の小文字で作成する。

- 単語はハイフンで区切る
- 日付は原則入れない
- 地域、会場、イベント種別の一部を使用する
- 長くなりすぎない
- 日本語や意味のない文字列は避ける

例：

kkr-osaka-resident-magic-show

## 禁止事項

- 入力されていない人数や年齢を作る
- お客様の発言を創作する
- 会場設備を確認せずに記載する
- 担当マジシャンの経歴を創作する
- 「絶対に成功します」などの断定表現
- 他社を下げる表現
- マジックの種明かし
- 個人が特定される情報の無断掲載

## 過去記事の分析（5本の実績記事に共通する特徴）

> 以下は examples/ の5記事を分析して抽出した、実際の記事に共通する良い特徴。
> 記事ごとの表現のばらつきではなく、安定して共通しているパターンを優先して記載。

### タイトルの付け方

- 形式は全記事で統一：【地域】会場名でイベント名のマジックショーを行いました～担当者名～
- 複数人の場合は「～滝沢涼、まさゆき、慶匠～」のように読点（、）区切り
- 地域は市区レベル（大阪市、大阪市浪速区、神戸市長田区、姫路市など）
- 地域に補足がある場合は「神戸市･有馬温泉」のように中黒で併記

### 冒頭文

- 「こんにちは。株式会社なんで屋のまさゆきです！」で統一（一部「こんにちは！」と感嘆符が先に来る記事もあるが、基本は句点のあとに名前）
- 冒頭2～3文で「いつ・どこで・何のイベントで」を簡潔に説明
- 来場者数と年齢層に軽く触れる
- 「こちらの出張マジックを報告します！同じようなイベントを検討中の方の参考になれば嬉しいです＾＾」が全記事で共通する定型文

### 文章の長さ

- 全体で読了時間3～4分（およそ1500～2500文字程度）
- 各見出しセクションは2～5文程度で簡潔
- マジックショーの内容セクションが最も長く、全体の約3～4割を占める
- 1文は短め（40～80文字程度）で読みやすさを重視

### 見出し構成

- 基本構成：イベント概要 → タイムスケジュール → 担当マジシャン → ショーの内容 → 演出のポイント → 会場・お客様の反応 → お問い合わせ案内
- タイムスケジュールがない記事もある（有馬温泉の記事ではイベント概要の中に含めている）
- 「演出のポイント」がない記事もある（省略されるのではなく、ショー内容に含まれることがある）
- ショーの内容の見出しは「ステージイリュージョンショーの内容」「サロンマジックショーの内容」「マジックショーの内容」とショー形式に合わせて変えている
- 最後のお問い合わせセクションは「関西圏で同じような○○をご検討中の方へ」が基本形

### 言葉遣い

- 「お届けしました」「披露しました」「盛り上がりました」が自然に使われる
- 「＾＾」を冒頭の定型文やポジティブな場面で使用（1記事に1～2回程度）
- 「！」は見出し的な短文やポイントの強調で使用。本文中では控えめ
- 「させていただきました」は少なく、「行いました」「お届けしました」が中心
- 「～ですよ！」「～でした＾＾」など、読者に語りかける柔らかい口調が時折入る
- 語尾の繰り返しを避け、「ました」「です」「ていました」を混ぜている

### マジック内容の説明方法

- 演目ごとに段落を分けて、時系列で「まずは→続いて→その後→中盤→後半→ラスト」と流れで説明
- 各演目に「誰が」「何を使って」「どんなマジックを」の3要素を含める
- 演目の説明のあとに、お客様の反応や会場の雰囲気を1文添える
- 写真キャプション的な短い1文（「迫力満点の人体切断イリュージョン！」など）を演目説明の間に挟む
- 仕掛けや種明かしには一切触れない
- 箇条書きではなく文章として自然につなげるのが基本（有馬温泉の記事のみ箇条書き形式）

### お客様の反応の書き方

- 「特に反応が大きかったのは○○でした」と具体的な演目を名指し
- 「驚きの声」「拍手」「見とれていた」など、実際に観察できる反応を使う
- 全体的な雰囲気（「終始あたたかく」「お祝いらしい明るい雰囲気」）も添える
- お客様の具体的な発言は創作せず、行動ベースで描写
- 記事によっては「会場・お客様の反応」が独立セクション、または演出のポイントに含まれる

### お問い合わせへのつなげ方

- 見出し：「関西圏で同じような○○をご検討中の方へ」（○○はイベント種別に合わせる）
- まず「今回のように、」で始め、箇条書きで対応可能なイベント例を4つ程度列挙
- 箇条書きの内容は、今回の事例に関連するキーワードを含める
- 締めは「会場の広さや客層、ご希望の雰囲気に合わせてショー内容をご提案できますので、ぜひお気軽にご相談ください。」が共通の定型文
- 最後に「以上、まさゆきでした！」で記事本文を締める
- その後、共通フッター（株式会社なんで屋にお任せください！！〜問い合わせフォーム案内）が続く

### 共通フッター

全記事の末尾に以下の定型文が共通で付く。

> イベントのマジックショーは株式会社なんで屋にお任せください！！
>
> 大切なイベントにプロのパフォーマーを派遣いたしませんか？
> 企業の周年パーティ、展示会の顧客満足度向上、ホームパーティなど
> どの場面でもご利用いただけます！
>
> 人数規模に合ったパフォーマンスをいたします！
> 少人数から大人数まで対応可能ですのでお気軽にお問い合わせください！
>
> ご相談は「お問い合わせフォーム」or「LINE」にてお気軽にどうぞ！

### URLスラッグの実例パターン

| 記事 | スラッグ |
|------|----------|
| KKRホテル大阪・懇親会 | osaka-family-social-gathering-magic-show |
| ホテルモントレ・企業パーティ | osaka-corporate-party-illusion-show |
| 有馬温泉さくら祭 | arima-sakura-festival-magic |
| 兵庫三菱・1周年イベント | kobe-nagata-anniversary-magic-show |
| 和ダイニング三日潮・創立記念 | himeji-anniversary-party-illusion-show |

共通ルール：
- 地域（市区または地名）を先頭に置く
- イベント種別を英語で表現
- ショー形式がイリュージョンの場合は「illusion-show」、通常は「magic-show」や「magic」
- 会場名は入れる場合と入れない場合がある（長くなりすぎる場合は省略）
- 5～7単語程度が目安
